# asteroids
A simple monochrome asteroid clone written in python.

## Usage
Once cloned, run `python main.py` to start the game.

Use the `A` and `D` keys to steer the ship and the `W` key to shoot.
Use the `SPACE` key to move the ship in the direction you are currently facing.

## Dependencies
The only dependency is `pygame` (can by installed with `pip install pygame`).

## Configuration
File `utils.py` contains some basic configuration.
To change the window size, you can change `SCREEN_WIDTH` and `SCREEN_HEIGHT`.
Change `SCALE` to control the size of the game field.
You can also set `COLOR_ON` and `COLOR_OFF` to change the color combo.
